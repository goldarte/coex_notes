## Полезные сервисные команды Linux

```bash
# Список томов с их размерами и заполнением
df -h
# Список файлов и папок с размерами в корне текущей директории
du -h .
# Отсортированный список файлов и папок с размерами по убыванию
du -hs | sort -hr
# Создать файл test размером 5000 Мб в текущей директории
dd if=/dev/zero of=test bs=1M count=5000
```
